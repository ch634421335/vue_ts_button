# vuets

### 简介

本项目是个人练习 vue2+ts 写的 button 组件。  
添加 xsmall,small,large,xlarge 调整大小  
添加 tile,radius,rounded 调整圆角  
添加 disabled 禁用按钮  
添加 color=''改变颜色

示例图
